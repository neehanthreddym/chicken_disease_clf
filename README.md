# Chicken disease classifier
CNN-based image classification system for identifying chicken diseases using deep learning and computer vision techniques.

## Workflow
1. **Update `config.yaml`**  
   Modify system configuration settings.  
2. **Update `secrets.yaml` (Optional)**  
   Add sensitive credentials like API keys.  
3. **Update `params.yaml`**  
   Adjust parameters for model training/testing.  
4. **Update the Entity**  
   Modify data entities for accurate input/output.  
5. **Update the Configuration Manager in `src/config`**  
   Adjust configurations for data pipeline and model.  
6. **Update the Components**  
   Modify or add components like feature extractors.  
7. **Update the Pipeline**  
   Update the processing steps for smooth execution.  
8. **Update `main.py`**  
   Modify the main script to align with updates.  
9. **Update `dvc.yaml`**  
   Track dataset and model changes using DVC.
